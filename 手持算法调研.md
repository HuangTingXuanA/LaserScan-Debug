# 设计的匹配算法
基于激光线连通区域的点重投影后限制在D_thresh_做匹配 -> 基于左配右和右配左的区间进行区间分割（双向验证分割，处理遮挡和假连通）
-> 基于分割的区间进行Census代价+几何特征计算综合得分 -> 使用LCS线序算法进行最终的一对一区间匹配
